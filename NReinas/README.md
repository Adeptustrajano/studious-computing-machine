# NQueens
Juego de las NREINAS para ensamblador ARMv4Keil Uvision5, según las dimensiones del tablero te calcula todas las posibles soluciones Nreinas posicion en un tablero NxN.
Diseñado por mi en asignatura de Arquitectura de Computación Ingenería Informática U.Politécnica. Zaragoza
